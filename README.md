## Desafio 1: 

- *Com suas palavras defina o que é um sistema de controle de versões (VCS)?  (máx 10 linhas):*


    R- São sistemas que auxiliam times de desenvolvimento fazer a gestão de mudanças e versionamento no código-fonte em todo
       ciclo de vida de projeto. 



- *Cite 5 vantagens em utilizar um VCS:*

1. Gerenciamento de alterações no código-fonte.
2. Rastreabilidade da evolução do projeto.
3. Colaboração simultânea em time.
4. Ramificação e mesclagem de código do projeto.
5. Redução de erros e duplicidades no projeto.

- *Cite 3 exemplos de VCS:*

1. Git
2. SVN
3. CVS
