## Desafio 5 :

- **Defina Entity :**

    > R - É uma representação do domínio do banco de dado em forma de classes que serão
  > mapeadas através de uma ORM e persistidas na camada de dados .


- **Defina Controller :**
    
    > R - Controller ou camada de controle é responsável por intermediar as demais
  > camadas da aplicação através das requisições.

- **Defina Repository :**
    > R - Repository é um padrão (Design Pattern) na qual fornece interface de acesso
  > a camada de dados permitindo o encapsulamento da lógica de acesso aos dados.

- **Defina Service :**
    > R - A Camada de serviço é responsável por intermediar a comunicação entre um
  > controller e uma interface repository, bem contém toda a lógica da regra de négocio
  > por trás das requisições chamadas pelo controller.
