# Desafio 6  - DTO:

- **O que significa DTO, e qual sua importância? :**
    > R - DTO é um acrônimo para Data Transfer Object (Em tradução livre: Objeto de Transferência de dados)
  > que é um padrão de projeto (Design Pattern) que consiste em um objeto ou um conjunto de objetos utilizado para tranferencia de
  > dados entre as camadas da web API, atende essencialmente ao propósito de transferência de dados entre as camada de uma aplicação
  > sem que uma informação seja divulgada para o Front-End desnecessariamente, resultando numa maior segurança na aplicação.
